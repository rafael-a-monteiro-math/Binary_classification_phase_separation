1 Introduction

This is a short companion website to the article Binary Classification as a Phase Separation Process, by Rafael Monteiro. A preprint is available on arXiv.

…but, before we start…

  • The main module for this project is binary_phase_separation.py, that you will find in this Github page (Monteiro 2020a).

  • If you are looking for the dataset and trained models, see the data repository at Zenodo.

  • If you want to reproduce this notebook you should download the file PSBC_Examples.tar.gz in the aforementioned data repository, and also this tarball in the Github page (Monteiro 2020b)for this project.1

  • To download data, I highly recommend the use of the script download_PSBC.sh, for a few reasons: it downloads the data in a safe way (using wget), and it automatically checks the MD5SUM of each file. We would like to highlight that you can also use this script to download only the trained examples and only the jupyter-notebooks for this project.

  • If you want to use the script download_PSBC.sh to download the dataset, please read first this README.pdf guide for further explanation (pages 3 and 4).

  • This website is based on this jupyter-notebook. The full code to the pictures seen here are posted there. For that reason, I shall refrain from posting the whole oputput of some cases, marking then with a symbol \(>>>\).

  • We will summarize some of the outputs, adding a “…” to them. If you would like to see the whole output, please see Notebook_PSBC_example.ipynb.

As I said, this discussion is short, so we will point out references along the way.
